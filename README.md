-> 2. Architecture write up
----
    2.1 Main page interfaces were created for navigating accross the multiple options a user could face for ordering food.
    2.2 Under each navigatable page the html code was written to incorporate the main requiremnts for the webpage.
    2.3 Javascript code was initiated for creating a product portfolio which can be easily managed by an admin.
    2.4 javescript was then developed to create a registration, login and logout interface for the user to be able to order food.
    2.5 A Cart was then created in java script to be able to handle orders and how items would be checked out.
    2.6 Final items were added as needed to be able to create a more friendlier interface wutg the users.

-> 3. Project User Stories
----
    3.1 As an admin I want to be able to have a website that can manage menu items, handle user orders, and handle payments and carts.
    3.2 As a user I want to be able to order menu items, load to a cart and check out.
